# interviews

This is a CLI program to run and check interview problems.

## Environment

- **Set API key:** Create a `.env` file at the project root (or copy `.env.template`) and set `SERPAPI_API_KEY` to your SerpApi key.
- **Install deps:** `pipenv install` (or add `python-dotenv` to your environment) so scripts can load `.env` automatically.

Example `.env` (do NOT commit your real key):

```
SERPAPI_API_KEY="your_serpapi_api_key_here"
```
